## 1. 功能清单

### 1.1 核心功能总览

| 功能名称 | 端点/方式 | 适用场景 | 实时性 |
|---------|----------|---------|--------|
| 异步语音合成 | `POST /v1/t2a_async_v2` | 长文本配音、批量处理 | 非实时 |
| 同步语音合成 | `WebSocket /ws/v1/t2a_v2` | 实时对话、流式输出 | 实时 |
| 音色克隆 | `POST /v1/voice_clone` | 自定义音色创建 | 非实时 |

### 1.2 功能详细说明

#### 1.2.1 异步语音合成 (speech-t2a-async)

**端点**: `POST /v1/t2a_async_v2`

**核心特性**:
- 支持6种模型: `speech-2.8-hd`, `speech-2.6-hd`, `speech-2.8-turbo`, `speech-2.6-turbo`, `speech-02-hd`, `speech-02-turbo`
- 单文件文本限制: **<100,000字符**
- 返回音频下载URL，有效期 **9小时**
- 支持时间戳（字幕）返回
- 支持发音字典（pronunciation_dict）
- 支持声音特效（sound_effects）

**流程阶段**:
```
[阶段1] 文本输入 → 获取file_id (若传text则跳过)
    ↓
[阶段2] 创建任务 → 获取task_id
    ↓
[阶段3] 轮询任务状态 (polling)
    ↓
[阶段4] 任务完成 → 获取file_id
    ↓
[阶段5] 下载音频文件
```

#### 1.2.2 同步语音合成 (speech-t2a-websocket)

**端点**: `wss://api.minimaxi.com/ws/v1/t2a_v2`

**核心特性**:
- WebSocket流式返回
- 实时播放，无需等待完整合成
- 文本限制: **<10,000字符**
- 需要本地安装 **mpv播放器**

**流程阶段**:
```
[阶段1] 建立WebSocket连接
    ↓
[阶段2] 发送开始任务指令
    ↓
[阶段3] 流式发送文本内容
    ↓
[阶段4] 接收音频流数据
    ↓
[阶段5] 保存音频文件 / 实时播放
```

#### 1.2.3 音色克隆 (speech-voice-clone)

**端点**: `POST /v1/voice_clone`

**核心特性**:
- 输入音频格式: mp3/m4a/wav
- 输入音频时长: **10秒 - 5分钟**
- 输入音频大小: **≤20MB**
- 可选输入示例音频: <8秒, ≤20MB
- 输出: custom voice_id + 试听音频

**流程阶段**:
```
[阶段1] 上传克隆音频 (10秒-5分钟, ≤20MB)
    ↓
[阶段2] 上传示例音频 (可选, <8秒, ≤20MB)
    ↓
[阶段3] 调用克隆接口
    ↓
[阶段4] 获得custom_voice_id
    ↓
[阶段5] 使用voice_id进行语音合成
```

## 2. 参数对照表

### 2.1 异步语音合成参数 (`/v1/t2a_async_v2`)

| 参数名 | 类型 | 取值范围 | 默认值 | 必填 | 说明 |
|-------|------|---------|-------|-----|-----|
| `text` | string | - | - | 否 | 直接传文本（与file_id二选一） |
| `file_id` | string | - | - | 否 | 上传文件ID（与text二选一） |
| `model` | string | 见下方模型列表 | - | 是 | 语音模型 |
| `voice_id` | string | 见音色清单 | - | 是 | 音色ID |
| `speed` | float | 0.5 - 2.0 | 1.0 | 否 | 语速 |
| `pitch` | int | -12 - 12 | 0 | 否 | 音调 |
| `vol` | float | - | 1.0 | 否 | 音量 |
| `audio_sample_rate` | int | 32000, 16000等 | 32000 | 否 | 采样率 |
| `bitrate` | int | 128000, 64000等 | 128000 | 否 | 比特率 |
| `format` | string | mp3, wav等 | mp3 | 否 | 音频格式 |
| `return_timestamps` | bool | true/false | false | 否 | 是否返回时间戳（字幕） |
| `pronunciation_dict` | object | - | - | 否 | 发音字典 |
| `sound_effects` | string | 见声音特效列表 | - | 否 | 声音特效 |

**模型枚举值**:
```
speech-2.8-hd      # 高清模型，2.8版本
speech-2.6-hd      # 高清模型，2.6版本
speech-2.8-turbo   # 加速模型，2.8版本
speech-2.6-turbo   # 加速模型，2.6版本
speech-02-hd       # 高清模型，02版本
speech-02-turbo    # 加速模型，02版本
```

**声音特效 (sound_effects)**:
```
spacious_echo       # 空旷回声
cave_echo           # 洞穴回声
church_ambience     # 教堂环境音
robot               # 机器人
chipmunk            # 花栗鼠（变高音调）
```

### 2.2 发音字典 (pronunciation_dict) 结构

```json
{
  "dict": [
    {
      "word": "自定义词",
      "phoneme": "zi4 ding4 yi4 ci2"
    },
    {
      "word": "MiniMax",
      "phoneme": "mei ni mei ke si"
    }
  ]
}
```

### 2.3 音色克隆参数 (`/v1/voice_clone`)

| 参数名 | 类型 | 取值范围 | 默认值 | 必填 | 说明 |
|-------|------|---------|-------|-----|-----|
| `audio_file` | file | mp3/m4a/wav | - | 是 | 克隆源音频 |
| `reference_audio` | file | mp3/m4a/wav, <8s | - | 否 | 参考示例音频 |
| `custom_voice_name` | string | - | - | 否 | 自定义音色名称 |

### 2.4 任务状态查询参数

| 状态值 | 说明 |
|-------|------|
| `Pending` | 任务排队中 |
| `Processing` | 正在处理 |
| `Success` | 处理完成 |
| `Failed` | 处理失败 |

## 5. 技术限制

### 5.1 文本长度限制

| 合成方式 | 最大字符数 | 说明 |
|---------|-----------|------|
| 异步合成 | **100,000字符** | 单个文本文件 |
| 同步合成 | **10,000字符** | WebSocket单次发送 |

### 5.2 文件限制

| 类型 | 限制 | 说明 |
|------|------|------|
| 克隆源音频 | 10秒 - 5分钟, ≤20MB | mp3/m4a/wav |
| 参考示例音频 | <8秒, ≤20MB | mp3/m4a/wav (可选) |
| 异步合成输入 | <10万字符 | text或file |

### 5.3 音频参数限制

| 参数 | 取值范围 | 默认值 | 单位 |
|------|---------|-------|-----|
| 语速 (speed) | 0.5 - 2.0 | 1.0 | 倍率 |
| 音调 (pitch) | -12 - 12 | 0 | 音高单位 |
| 采样率 | 32000, 16000等 | 32000 | Hz |
| 比特率 | 128000, 64000等 | 128000 | bps |

### 5.4 时间限制

| 项目 | 限制 | 说明 |
|------|------|------|
| 音频下载URL有效期 | **9小时** | 异步合成完成后 |
| 克隆处理时间 | 约1-3分钟 | 视音频复杂度 |
| 轮询间隔建议 | 3-5秒 | 避免请求过于频繁 |

### 5.5 并发与配额限制

| 限制类型 | 限制值 | 说明 |
|---------|-------|------|
| 单账号并发任务数 | 视套餐 | 避免同时创建过多任务 |
| API调用频率 | 视套餐 | 建议添加重试机制 |
| WebSocket连接数 | 视套餐 | 同步合成时 |

### B. 错误码参考

| 错误码 | 说明 | 处理建议 |
|-------|------|---------|
| `text_too_long` | 文本超出限制 | 缩短文本或分段合成 |
| `invalid_voice_id` | 音色ID不存在 | 检查音色ID或刷新音色列表 |
| `model_not_found` | 模型不存在 | 检查model参数值 |
| `file_expired` | 文件已过期 | 重新上传或创建任务 |
| `task_failed` | 任务处理失败 | 查看详细错误信息或重试 |
| `rate_limit_exceeded` | 请求过于频繁 | 降低请求频率 |
| `invalid_audio_format` | 音频格式不支持 | 转换音频格式 |
